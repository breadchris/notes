8529 people have contributed to scikit-learn

March 26, 2024| breadchris
<p>The code that scikit-learn comprises has been committed to by 8529+ people.</p> <ul> <li> <p><a href="https://github.com/python/cpython">python/cpython</a> - 2640</p> </li> <li> <p><a href="https://github.com/numpy/numpy">numpy/numpy</a> - 1581</p> </li> <li> <p><a href="https://github.com/scipy/scipy">scipy/scipy</a> - 1447</p> </li> <li> <p><a href="https://github.com/joblib/threadpoolctl">joblib/threadpoolctl</a> - 10</p> </li> <li> <p><a href="https://github.com/scikit-learn/scikit-learn">scikit-learn/scikit-learn</a> - 2837</p> </li> </ul> <p>It is possible that some of these people introduced typo changes, or the code they contributed has been rewritten by someone else. But the repo was affected by them, no less. Their accounts were authenticated to contribute to the history of a notable library in the ML training community. The more valuable ML becomes, the greater the desire to influence its capabilities. Management of this code is split between five different repos across five organizations.</p> <p>This is only one training library; the ML community uses an ever-growing number of libraries.</p> <p>What about the drivers used to perform the GPU operations on the data? The operating system. What about the base model or training data from huggingface? The data scientist trains the model on their beefy laptop in other applications. Who has access to them, and what code could they sneak in? What if their account was hacked, and someone else made changes?</p> <p>We are cresting on a significant surge of value from advancements in ML. Influencing how AI performs could mean changing how <a href="https://resources.data.gov/resources/fdspp-nih-leveraging-ai-for-business-process-automation/">significant decisions</a> are made. I illustrated the state of one threat vector in an expanding attack surface.</p> <p>The security recommendations for securing an ML pipeline are still early. Most of the focus of AI security is on adversarial attacks, often overlooking the process of building a model. It is a cross-disciplinary effort between data scientists, software engineers, and infrastructure.</p> <p>For such a significant achievement in human advancement, I think it is essential to map out this problem space more and focus on securing this infrastructure.</p>
